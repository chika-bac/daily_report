## わかったこと

- Ruby のクラスからインスタンス作成時に引数を入れておき、initialize メソッドの引数に指定・メソッド内でその値を代入しておくと、インスタンスごとにインスタンス変数の値を変えることができる
  ```
  def initialize(name:,price:)
    self.name = name
    self.price = price
  end
  ```
- initialize メソッドのオーバーライドは、親クラスの内容を`super()`とすることで継承できる
